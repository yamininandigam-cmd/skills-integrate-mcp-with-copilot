# Add Skills badges and Coding-Profile links

Description
-----------
Add a section that displays skills as visually distinct badges and includes links to common coding profiles (HackerRank, LeetCode, CodeChef, Codeforces, InterviewBit). Optionally show small rating/level indicators.

Acceptance criteria
-------------------
- `#abilities` section added to `src/static/index.html` listing skills in badge style.
- Links to at least two sample coding profile URLs included.
- CSS added in `src/static/styles.css` for badges and star/level visuals.

Suggested files to update
-------------------------
- `src/static/index.html` (add `#abilities` block)
- `src/static/styles.css` (badge styles)

Notes
-----
Badges may be implemented with simple styled `<span>` elements; no backend changes required.
