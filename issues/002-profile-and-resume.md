
# Add Profile / Resume sections and contact details

Description
-----------
Expand the static frontend with a `Profile` / `Resume` area that includes: About, Education, Experiences, Certifications, and Contact information. Provide links to downloadable resume and hosted certificates/demos.

Acceptance criteria
-------------------
- New `#profile` section in `src/static/index.html` with personal details and an image placeholder.
- Sections for Education, Internships, Certifications, and Contact with example content.
- Contact links (email, LinkedIn) visible in the Contact area.

Suggested files to update
-------------------------
- `src/static/index.html` (add profile/resume content and anchors)
- `src/static/styles.css` (styling for resume sections)

Notes
-----
Content can be anonymized or use a sample template; this is for UX completeness rather than user-specific data.
